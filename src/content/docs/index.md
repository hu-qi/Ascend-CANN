## 接下来可以做什么

- 更新内容：编辑 `src/content/docs/index.md` 立即查看首页的变化。
- 调整布局：删除 `template: splash` 即可在此页面显示侧边栏。
- 添加新页面：在 `src/content/docs` 中新增 Markdown 文件生成更多页面。
- 配置站点：在 `astro.config.mjs` 中更新侧边栏与其他设置。
- 深入了解：查阅 [Starlight 文档](https://starlight.astro.build/) 获取更多信息。
